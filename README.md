# Genome::Genome

Represent Bacterial genomes in Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'genome'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install genome
```
